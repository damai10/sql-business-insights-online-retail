# 📊 Business Insights using SQL (Online Retail Dataset)

## 📌 Overview
This project focuses on extracting meaningful business insights from transactional data using SQL.

The analysis covers revenue trends, customer behavior, and product performance to support data-driven decision-making.

---

## 🛠️ Tools Used
- MySQL
- SQL

---

## 🔍 Key Analysis
- Total Revenue & Orders  
- Monthly Revenue Trends  
- Top Customers  
- Revenue by Country  
- Product Performance  

---

## 📈 Key Insights
- Revenue shows variation across months indicating changing demand patterns  
- A small group of customers contributes significantly to total revenue  
- Certain products generate high sales and drive business performance  
- Sales are concentrated in specific countries
